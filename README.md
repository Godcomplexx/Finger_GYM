# Finger GYM

Finger GYM - прототип модуля предварительного тестирования мелкой моторики кисти с использованием веб-камеры и MediaPipe Hands.

Проект отслеживает 21 ключевую точку кисти, проводит калибровку по открытой ладони, выполняет серию тестовых упражнений и сохраняет результат сессии в JSON.

## Возможности

- трекинг кисти через MediaPipe Hand Landmarker;
- визуализация скелета руки поверх изображения с камеры;
- калибровка базовых параметров ладони;
- упражнения для оценки открытой ладони, кулака, щипкового захвата, указательного жеста, поворота кисти, движения по зонам и удержания;
- расчет частных метрик и итогового индекса готовности;
- сохранение сессий в локальные JSON-файлы;
- модульные тесты для метрик, упражнений, скоринга и хранения.

## Установка

Требуется Python 3.10+ и веб-камера.

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

При обычном запуске сначала открывается экран выбора трекера: `1/M` или клик мышью - MediaPipe, `2/U` или клик мышью - Ultraleap.

Режимы трекинга:

```bash
python main.py --tracker mediapipe
python main.py --tracker ultraleap
```

`mediapipe` использует веб-камеру и модель `hand_landmarker.task`. `ultraleap` использует Ultraleap Hand Tracking Software и подключается напрямую к `LeapC.dll`, поэтому перед запуском должны быть установлены SDK/сервис Ultraleap.

В режиме Ultraleap обычного видео с камеры нет: приложение показывает виртуальную рабочую зону и рисует скелет руки по координатам датчика.

Для Ultraleap можно настроить рабочую область нормализации координат:

```bash
set ULTRALEAP_WORKSPACE_WIDTH_MM=300
set ULTRALEAP_WORKSPACE_HEIGHT_MM=300
set ULTRALEAP_WORKSPACE_CENTER_Y_MM=200
python main.py --tracker ultraleap
```

Дополнительные параметры:

```bash
python main.py --camera 0 --patient patient-001 --width 1280 --height 720
```

Управление в приложении:

- `R` - выбрать правую руку;
- `L` - выбрать левую руку;
- на экране выбора руки можно кликнуть мышью по кнопке или навести указательный палец на кнопку и подержать;
- `Q` или `Esc` - выйти.

## Данные

Результаты сессий сохраняются локально в папку `sessions/`. Эта папка исключена из git, потому что может содержать данные конкретных запусков.

## Примечание

В текущей реализации проект использует экранные 2D-координаты MediaPipe (`x`, `y`). Глубина (`z`) и `world_landmarks` пока не сохраняются и не участвуют в расчете метрик.
