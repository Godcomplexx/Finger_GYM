# Реестр рисков

| ID | Риск | Причина | Последствие | Мера управления | Остаточный статус |
|---|---|---|---|---|---|
| R-01 | Неверная рекомендация при плохом трекинге | Потеря руки, фон, освещение | Неправильный сценарий реабилитации | Порог `validTrackingRatio`, статус `unreliable`, повтор теста | Требует испытаний |
| R-02 | Интерпретация результата как диагноза | Некорректное позиционирование продукта | Медицинская ошибка | Текст "не заменяет специалиста", обязательное подтверждение | Требует включения в РЭ/UI |
| R-03 | Ошибка при калибровке | Недостаточно валидных кадров | Систематическая ошибка метрик | Отказ от рекомендации при плохой калибровке | Частично реализовано |
| R-04 | Несанкционированный доступ к данным | Локальные JSON-файлы без контроля доступа | Раскрытие ПДн | Обезличивание, роли, права ОС/АПК, журнал доступа | Требует ИБ-реализации |
| R-05 | Неконтролируемое изменение модели | Замена `.task` файла | Невоспроизводимые результаты | SHA-256 модели в протоколе | Частично реализовано |
| R-06 | Невалидированные клинические пороги | Пороги выбраны инженерно | Ошибка классификации | Клиническая валидация и утверждение методистом | Не закрыто |
| R-07 | Смещение датасета | Недостаточно разнообразная выборка | Низкое качество на целевой группе | Паспорт датасета, внешний тест | Не закрыто |
| R-08 | Сбой интеграции с АПК | Неполный обмен данными | Потеря результата | JSON spec, интеграционные тесты | Требует реализации API |
| R-09 | Некорректная сторона руки | Ошибка выбора правой/левой руки | Ошибка в упражнениях поворота | Выбор руки оператором, журнал события, тесты | Частично реализовано |
| R-10 | Использование устаревшей версии алгоритма | Нет release control | Невоспроизводимость | `MODULE_VERSION`, `ALGORITHM_VERSION`, changelog | Частично реализовано |
